## taro-plugin-config

小程序项目配置插件，可以生成对应小程序的项目配文件，有任何问题可以 [issues](https://github.com/yangtianxia/tarojs-template/issues)

> 注意：该插件可能会与 `tarojs` 自带的项目配置冲突，请避免在根目录添加小程序相关配置文件 [项目配置](https://taro-docs.jd.com/docs/project-config)
